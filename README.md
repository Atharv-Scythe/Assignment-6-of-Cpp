# C++ Programming Assignment – 6

## 📌 Description
This repository contains solutions to **Assignment 6 of C++ Programming**.  
The assignment focuses on **advanced C++ concepts** such as variables, pointers, functions, parameter passing, function overloading, and arrays.

The goal is to build a strong understanding of **memory, functions, and program behavior**.

All programs are written in **C++ (.cpp)** and compiled using **g++ (GCC compiler)**.
